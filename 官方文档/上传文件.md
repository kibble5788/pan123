 # 创建文件接口说明
## 一、接口基础信息
- **API 地址**：POST 域名 + /upload/v2/file/create
- **核心说明**：
  1. 文件名限制：小于 256 个字符，且不能包含字符："\/:*?|><，同时不能全部是空格。
  2. 上传大小限制：开发者单文件上传最大支持 10GB。

## 二、请求参数
### （一）Header 参数
| 名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| Authorization | string | 是 | 鉴权 access_token |
| Platform | string | 是 | 固定值：open_platform |

### （二）Body 参数
| 名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| parentFileID | number | 是 | 父目录 ID，上传到根目录时填写 0 |
| filename | string | 是 | 文件名小于 255 个字符，不能包含字符："\/:*?|><，且不能重名；containDir 为 true 时，需传入路径+文件名（示例：/你好/123/测试文件.mp4） |
| etag | string | 是 | 文件 MD5 |
| size | number | 是 | 文件大小，单位：byte（字节） |
| duplicate | number | 否 | 相同文件名处理策略：1-保留两者（新文件自动加后缀），2-覆盖原文件 |
| containDir | bool | 否 | 上传文件是否包含路径，默认值：false |

## 三、返回数据
| 名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| fileID | number | 否 | 文件 ID，仅秒传（文件已存在）时返回，唯一标识文件 |
| preuploadID | string | 是 | 预上传 ID，reuse 为 true 时该字段不存在 |
| reuse | boolean | 是 | 是否秒传，返回 true 表示文件已上传成功 |
| sliceSize | number | 是 | 分片大小，需按此大小生成文件分片后上传 |
| servers | array | 是 | 上传地址列表 |

## 四、请求示例（Curl）
```curl
curl --location 'https://open-api.123pan.com/upload/v2/file/create' \
--header 'Content-Type: application/json' \
--header 'Platform: open_platform' \
--header 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJl...(过长省略)' \
--data '{
"parentFileID": 0,
"filename": "测试文件.mp4",
"etag": "4b5c549c4abd0a079caf92d6cad24127",
"size": 50650928
}'
```
