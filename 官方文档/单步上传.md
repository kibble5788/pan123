# 单步上传接口说明
## 一、接口基础信息
- **API 地址**：POST 上传域名 + /upload/v2/file/single/create
- **核心说明**：
  1. 文件名限制：小于 256 个字符，且不能包含字符："\/:*?|><，同时不能全部是空格。
  2. 上传大小限制：开发者单文件上传最大支持 1GB。
  3. 上传域名来源：需使用「获取上传域名」接口响应中返回的域名。
  4. 适用场景：小文件上传，一次 HTTP 请求即可完成上传全流程。

## 二、请求参数
### （一）Header 参数
| 名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| Authorization | string | 是 | 鉴权 access_token |
| Platform | string | 是 | 固定值：open_platform |

### （二）Body 参数（form-data 格式）
| 名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| parentFileID | number | 是 | 父目录 ID，上传到根目录时填写 0 |
| filename | string | 是 | 文件名小于 255 个字符，不能包含字符："\/:*?|><，且不能重名；containDir 为 true 时，需传入路径+文件名（示例：/你好/123/测试文件.mp4） |
| etag | string | 是 | 文件 MD5 |
| size | number | 是 | 文件大小，单位：byte（字节） |
| file | file | 是 | 文件二进制流 |
| duplicate | number | 否 | 相同文件名处理策略：1-保留两者（新文件自动加后缀），2-覆盖原文件 |
| containDir | bool | 否 | 上传文件是否包含路径，默认值：false |

## 三、返回数据
| 名称 | 类型 | 是否必填 | 说明 |
| ---- | ---- | ---- | ---- |
| fileID | number | 是 | 文件唯一 ID，秒传（文件已存在）时也会返回 |
| completed | bool | 是 | 是否上传完成，返回 true 表示上传成功 |

## 四、请求示例（Curl）
```curl
curl --request POST \
--url https://openapi-upload.123242.com/upload/v2/file/single/create \
--header 'Authorization: Bearer eyJhbGciOiJIUzI1...(过长省略)' \
--header 'Platform: open_platform' \
--header 'content-type: multipart/form-data' \
--form 'file=@C:\Users\mfy\Downloads\测试.exe' \
--form parentFileID=11522394 \
--form 'filename=测试.exe' \
--form etag=511215951b857390c3f30c17d0dae8ee \
--form size=35763200
```
